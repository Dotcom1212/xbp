テーマ

時間を調べると、アメリカニューヨークとニュージーランドのオークランドの時刻を日本の時刻と合わせて提示してくれる。

コード

            def get_local_time(city):
                timezones = {
                    '日本': 'Asia/Tokyo',
                    'アメリカ': 'America/New_York',
                    'オークランド': 'Pacific/Auckland'
                }
            
                if city not in timezones:
                    return "対応していない都市です"
                utc_now = datetime.utcnow()
                local_timezone = pytz.timezone(timezones[city])
                local_time = utc_now.replace(tzinfo=pytz.utc).astimezone(local_timezone)
                return local_time.strftime("%Y-%m-%d %H:%M:%S %Z")
            
            ユーザーに都市を尋ねる
            user_city = input("都市を選んでください（日本、アメリカ、オークランド）: ")
            
            ローカル時間を取得して表示
            result = get_local_time(user_city)
            print(result)
